# task6-product-catalog

# 🛒 Product Catalog Web App (Task 6 - Full Stack Internship)

This project is developed as part of **Maincrafts Technology Full Stack Web Development Internship - Task 6**.

In Task-6, the main goal is to connect the Product Catalog application to a **real backend API** using **Fetch API** and **async/await**, replacing the local JSON file used in Task-5.

---

## 📌 Features Implemented

✅ Fetch Products from Real API (GET Request)  
✅ Loading Spinner while fetching data  
✅ Error Message UI if API call fails  
✅ Search Products by Title  
✅ Filter Products by Category  
✅ Sort Products by Price and Title  
✅ Pagination Support  
✅ Add to Cart (LocalStorage)  
✅ Cart Count Display  
✅ Add New Product using POST Request  
✅ Delete Product using DELETE Request  
✅ Service Layer (`api.js`) for clean architecture

## ⚙️ How to Run the Project

### Method 1: Using Live Server (Recommended)

1. Download and install **VS Code**
2. Install the extension **Live Server**
3. Open the project folder in VS Code
4. Right click on `index.html`
5. Select **Open with Live Server**
